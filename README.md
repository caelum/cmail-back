## API
`http://localhost:3000/`  

Response formats: JSON

### Endpoints:
#### Users `/users`
**Accepts:**  
    - `GET`  
    - `POST`  

#### Parameters `user/:userId`
**Accepts:**  
    - `GET`  
    - `PATCH`   
    
#### Login `/login`
**Accepts:**  
    - `POST`
